# CS411-Spring-2023
## .env file in the server directory should have the following
```
PORT= <Port Number>
MONGO_URI= <For Mongo DB>
SECRET= < key for hash (can be any password)>
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_CLIENT_REDIRECT_URI=
KEY= <sessions key>
YOUTUBE_API_KEY=
```
